# Slotify Music
A music streaming app.


## Technologies Used


 - PHP
 - Javscript
 - HTML
 - CSS
 
 ## Features
 
 
 
 - Login and signup
 - Listen to songs
 - Music player like interface , slide the seekbar, volumebar etc
 - Mute & Unmute
 - Next and Previous song
 - Playlist
 - Create Playlist
 - Shuffle Playlist
 - Popular songs and view count
 - Songs by Artist
 - Songs in Album
 - Search by Name, Album or Artist
