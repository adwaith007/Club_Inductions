# Cryptocurrency
A digital wallet implementation and an implementation of blockchain concept.


## Technologies Used


 - NodeJs
 - Express
 - Javscript
 - Jest testing package
 - Sockets
 
 ## Features
 
 
 
 - Data encrypted and stored in a block.
 - Block-chain implementation using 
 - Nonce (Proof of work)
 - Dynamically adjusting difficulty depending upon rate of mining.
 - Wallet
 - Transaction pool
 - Wrap unmined transactions by single user into single block to prevent double spending 
 - Mining transaction from transaction pool
 - Reward transaction for miners
