# Note Todo Webapp
A webapp where you can store your notes and todos.


## Technologies Used


 - NodeJs
 - Express
 - Javscript
 - HTML
 - CSS
 
 ## Features
 
 
 
 - Add todos or notes.
 - Add pics to notes.
 - Sort by Importance, Created On or Last Edited On.
 - Search by label.
 - Edit or delete notes or todos.
 - Password recovery options.
